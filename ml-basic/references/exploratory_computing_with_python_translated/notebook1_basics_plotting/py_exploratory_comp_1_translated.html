Python 入门第一步#

“便携、强大且易于使用”, Python 是一种流行的开源编程语言，既可用于编写脚本应用程序，也可用于独立程序（参见 Mark Lutz 的《Learning Python》）。Python 几乎可以用来做任何事情。例如，你可以把 Python 当作计算器使用。将光标放在下面的代码单元格中，按下 [shift][enter]。输出应该是 12 :-)

6 * 2

12

请注意，添加额外的空格是为了增加代码的可读性。 2 * 3 的效果和 2*3 一样。并且它被认为是一个很好的代码风格。请在你的所有笔记中使用这个额外空格。

当你正在编程，你想要把你的值存储在变量里

a = 6
b = 2
a * b

12

‘a’ 和 ‘b’ 现在都是变量了，每个变量都有一个类型。在这个例子里，他们都是整数类型。想要把变量的值输出在屏幕上，使用 print 函数（如果代码单元的最后一条语句没有存储在变量中，则会自动打印到屏幕上，如上所示）。注意，两个整数相乘得到的是整数，而两个整数相除得到的是浮点数（带有小数点的数字）。 、

print(a)
print(b)
print(a * b)
print(a / b)

6
2
12
3.0

你可以通过将文本字符串放在引号中（单引号或双引号都可以，只要开头和结尾使用相同的引号），然后用逗号分隔文本字符串和变量，来为 print 函数添加一些文本。

print('the value of a is', a)

the value of a is 6

变量可以通过使用 ** 进行幂运算（在某些其他语言中使用的 ^ 符号无法用于此操作）。

a ** b

除法#

除法也可以使用

print('1/3 gives', 1 / 3)

1/3 gives 0.3333333333333333

（针对 Python 2 用户的说明（你真的应该切换到 Python 3！）：在 Python 2 中，1/3 的结果为零，因为两个整数的除法在 Python 2 中返回一个整数。）上面的打印语句在连续显示 16 个 3 的值时看起来非常丑陋。更好且更易读的方法是使用称为 f-string 的格式来同时打印文本和变量的值。f-string 允许你通过用大括号 {} 包围变量的值，将其插入到文本的任何位置。整个文本字符串需要用引号括起来，并且前面需要加上字母 f，就像下面。

a = 1
b = 3
c = a / b
print(f'{a} divided by {b} gives {c}')

1 divided by 3 gives 0.3333333333333333

大括号中的完整语法是 {variable:width.precision}。当未指定 输出总长 和 保留小数位 时，Python 会使用所有数字并为你计算宽度。如果你想要一个小数点后有 3 位的浮点数，可以指定数字 3，后面跟上字母 f 表示浮点数（你仍然可以不指定宽度，让 Python 自动计算）。如果你更喜欢指数（科学）表示法，可以用 e 替换 f。 # 后面的文本是代码中的注释。在 # 后的任何文本都会被 Python 忽略。

print(f'{a} divided by {b} gives {c:.3f}') # 保留位小数
print(f'{a} divided by {b} gives {c:10.3f}') # 长度为10 保留三位小数
print(f'{a} divided by {b} gives {c:.3e}') # 三位小数用科学计数法

1 divided by 3 gives 0.333
1 divided by 3 gives      0.333
1 divided by 3 gives 3.333e-01

关于变量的更多信息#

一旦你在 Python 会话中创建了一个变量，它将保留在内存中，因此你可以在其他单元格中使用它。例如，两个代码单元格之前定义的变量 a 和 b 仍然存在。

print(f'the value of a is: {a}')
print(f'the value of b is: {b}')

the value of a is: 1
the value of b is: 1

用户（在这个例子中是你！）决定代码块的执行顺序。例如，In [6] 表示这是第六次执行一个代码块。如果你更改了同一个代码块并再次运行，它将编号为 7。如果你在代码块 7 中定义变量 a，它将覆盖之前代码块中定义的 a 的值。

变量名可以尽可能长（不过你得自己输入）。选择描述性的名称有助于理解代码。变量名不能包含空格，也不能以数字开头。并且变量名区分大小写。因此，变量 myvariable 与变量 MyVariable 是不同的。变量的名称可以是你想要的任何名称，但不能是 Python 语言中的保留字。例如，无法创建变量 for = 7，因为 for 是一个保留字。当我们继续学习时，你将了解许多保留字；在 Notebook 中输入时，它们会以粗体绿色显示。

让我们尝试绘制 \(y\) 相对于 \(x\) 的图，\(x\) 的范围从 \(-4\) 到 \(+4\)，对于多项式 \(y=ax^2+bx+c\)，其中 \(a=1\)，\(b=1\)，\(c=-6\)。为了做到这一点，我们需要在一系列点上评估 \(y\)。同一类型值的序列称为数组（例如整数数组或浮点数组）。数组功能在 numpy 包中可用。让我们导入 numpy 并将其命名为 np，这样可以将 numpy 包中的任何函数称为 np.function|

import numpy as np

要创建一个数组 x，例如包含 -4 和 4 之间 5 个等间距的点，可以使用 linspace 命令。

x = np.linspace(-4, 4, 5)
print(x)

[-4. -2.  0.  2.  4.]

在上面的单元格中，x 是一个包含 5 个浮点数的数组（-4. 是浮点数，-4 是整数）。如果你输入 np.linspace 然后打开一个左括号，如：

np.linspace(

请注意，在简单的语句 y = a * x ** 2 + b * x + c 中计算了 一百 个 y 值。当你进行数学运算时，Python 将数组视为与常规变量相同的方式处理。数学运算会简单地应用于数组中的每个值（并且运行速度比逐个计算要快得多）。

你可能会想知道像 [<matplotlib.lines.Line2D at 0x1de3ff61910>] 这样的语句是什么（你机器上的数字可能会有所不同）。这实际上是一个句柄，用于指向在代码块中的最后一个命令（在这个例子中是 plt.plot(x, y)）创建的线条。请记住：代码单元格中最后一行的结果会打印到屏幕上，除非它存储在变量中。你可以通过在代码块的最后一条命令后加上分号来告诉 Notebook 不要将其打印到屏幕上（例如输入 plot(x, y);）。稍后我们将学习将这个句柄存储在变量中也是有用的

plot 函数可以接受许多参数。通过输入 plt.plot( 然后按 shift-tab，你可以查看 plot 函数的帮助框，获得大量帮助。输入 plt.plot? 会在笔记本底部弹出一个新的可滚动子窗口，显示 plot 的文档。点击右上角的 x 可以关闭该子窗口。

简而言之，plot 可以用一个参数 plot(y) 来使用，这会在纵轴上绘制 y 值，并从 0 开始枚举横轴。plot(x, y) 绘制 y 相对于 x 的图，而 plot(x, y, formatstring) 则使用 formatstring 中定义的颜色和标记绘制 y 相对于 x 的图，formatstring 可以有很多形式。它可以用于定义颜色，例如 ‘b’ 表示蓝色，’r’ 表示红色，’g’ 表示绿色。也可以用于定义线型，’-’ 表示实线，’–’ 表示虚线，’:’ 表示点线。你还可以定义标记，例如 ‘o’ 表示圆形标记，’s’ 表示方形标记。甚至可以将它们组合在一起：’r–’ 表示红色虚线，而 ‘go’ 表示绿色圆形标记。

如果这还不够，plot 还接受大量的关键字参数。关键字参数是可以添加到函数的可选参数。语法为 function(keyword1=value1, keyword2=value2) 等等。例如，要绘制一条宽度为 6 的线（默认宽度为 1），可以输入：

plt.plot([1, 2, 3], [2, 4, 3], linewidth=6);

关键字参数应该放在普通参数之后。plot(linewidth=6, [1, 2, 3], [2, 4, 3]) 会报错。

可以使用 xlabel 和 ylabel 函数为坐标轴添加名称，例如 plt.xlabel(‘this is the x-axis’)。请注意，这两个函数都接受字符串作为参数。可以使用 plt.title 命令为图形添加标题。通过在同一个代码单元格中给出多个绘图命令，可以将多条曲线添加到同一图形中。它们会自动添加到同一图形中。

新图形和图形大小#

每当你在代码单元格中给出绘图语句时，都会自动创建一个默认大小的图形，所有后续的绘图语句都会添加到同一图形中。如果你想要不同大小的图形，可以先使用 plt.figure(figsize=(width, height)) 语法创建一个具有所需大小的图形。然后，代码单元格中的任何后续绘图语句都会添加到该图形中。你甚至可以创建第二个（或第三个、第四个……）图形。

plt.figure(figsize=(10, 3))
plt.plot([1, 2, 3], [2, 4, 3], linewidth=6)
plt.title('very wide figure')
plt.figure()  # new figure of default size
plt.plot([1, 2, 3], [1, 3, 1], 'r')
plt.title('second figure');

风格#

正如上面提到的，良好的编码风格非常重要。它使代码更易于阅读，从而更容易发现错误和漏洞。例如，考虑下面的代码，它重新创建了我们之前绘制的图形（使用更宽的线条），但现在没有插入额外的空格。（很挤很难看吧）

a=1
b=1
c=-6
x=np.linspace(-4,4,100)
y=a*x**2+b*x+c#Compute y for all x values
plt.plot(x,y,linewidth=3)

在前面的代码单元中，代码难以阅读。良好的风格至少包括以下几点：

  • 每个数学符号（=, +, -, *, /）周围都有空格，但 ** 周围不需要空格

  • 函数参数之间有空格

  • 关键字参数的等号两边没有空格（因此 linewidth=3 是正确的）

  • 每个逗号后有一个空格

  • 每个 # 后有一个空格

  • 当 # 紧跟在 Python 语句后面时，前面有两个空格

  • 函数名和参数列表之间没有空格。因此 plt.plot(x, y) 是良好的风格，而 plt.plot (x, y) 就不符合良好的风格。

这些规则是官方 Python 风格指南 PEP8 的（非常小的一部分）。当这些规则被应用时，代码可读性会大大提高，如下所示：

a = 1
b = 1
c = -6
x = np.linspace(-4, 4, 100)
y = a * x**2 + b * x + c  # Compute y for all x values
plt.plot(x, y, linewidth=3);

在所有其他练习和未来的 Notebook 中使用正确的风格。

颜色#

如果你没有为绘图语句指定颜色，matplotlib 将使用其默认颜色。前三种默认颜色是特定色调的蓝色、橙色和绿色。默认颜色的名称是一个大写的 C，后面跟着数字，从数字 0 开始。例如

plt.plot([0, 1], [0, 1], 'C0')
plt.plot([0, 1], [1, 2], 'C1')
plt.plot([0, 1], [2, 3], 'C2')
plt.legend(['default blue', 'default orange', 'default green']);

在 matplotlib 绘图中，有五种不同的方法可以指定你自己的颜色；你可以在这里了解更多。一个有用的方法是使用 HTML 颜色名称。你可以在以下链接找到 HTML 颜色代码的相关信息： 这里.

color1 = 'fuchsia'
color2 = 'lime'
color3 = 'DodgerBlue'
plt.plot([0, 1], [0, 1], color1)
plt.plot([0, 1], [1, 2], color2)
plt.plot([0, 1], [2, 3], color3)
plt.legend([color1, color2, color3]);

使用 xkcd 颜色名称可能是最酷（也是最呆板）的方法，这些名称需要以 xkcd: 前缀来指定。xkcd 颜色名称的列表可以在 xkcd 网站上找到，其中包括一些有趣的颜色，如“婴儿呕吐绿”和各种棕色，范围从“屎色”到“婴儿屎棕”。你可以尝试使用这些颜色名称来绘图，效果会很有趣！XD

plt.plot([1, 2, 3], [4, 5, 2], 'xkcd:baby puke green');
plt.title('xkcd color baby puke green');
